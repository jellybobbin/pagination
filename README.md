# Pagination

Pagination component for Vue Bulma.

## Installation

```sh
$ npm install vue-bulma-pagination --save
or
$ yarn add vue-bulma-pagination --save
```

## Examples

```vue
<template>
  <div>
    <pagination urlPrefix='/' currentPage=2 pageLength=100 /> 
  </div>
</template>

<script>
import Pagination from 'vue-bulma-pagination'

export default {
  components: {
    Pagination
  }
}
</script>
```
